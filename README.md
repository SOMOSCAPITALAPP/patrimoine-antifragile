# Patrimoine Antifragile

Prototype de landing page pour presenter une approche d'investissement antifragile autour de la marque personnelle `Pinko777`.

## Contenu

- `index.html` : page d'accueil / landing page
- `styles.css` : styles du prototype

## Positionnement

Le projet met en avant :

- l'investissement antifragile
- la robustesse patrimoniale
- la frugalite
- les interets composes
- une conversion douce vers l'univers de `Pinko777`

## Objectif

Construire un site media credible et pedagogique pouvant evoluer vers :

- une landing page plus avancee
- un mini-site multi-pages
- un blog SEO
- une vitrine pour un profil investisseur

## Lancer le prototype

Ouvrir `index.html` dans un navigateur.

## Prochaines etapes possibles

- ajouter une page `A propos`
- ajouter une page `Strategie antifragile`
- ajouter un formulaire relie a un outil d'emailing
- deplacer le site vers un framework moderne si besoin

## Note

Ce prototype a une vocation editoriale et informative. Il ne constitue pas un conseil en investissement personnalise.
